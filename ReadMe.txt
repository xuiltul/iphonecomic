iPod Touch用のイメージビューワーです。
Zipファイルでまとめたイメージを見る事ができます。

----
= 説明 =


== 操作方法 ==

ページの進め方は次の３つの方法があります。
  ・ボタン
  ・スワイプ
  ・重力

*ボタン*

  右下、左下で押すことで前のページ、次のページを表示します。
  進むか戻るかは、設定項目の「左ボタンで次のページ」で決定します。

  オンの場合、右下・・・次のページ。左下・・・前のページ。

  オフの場合、右下・・・前のページ。左下・・・次のページ。


*スワイプ*

  左、右にスワイプすることで、前のページ、次のページを表示します。
  進むか戻るかは、設定項目の「右にスライド」で決定します。

  オンの場合、右方向にスライド・・・次のページ。左方向にスライド・・・前のページ。

  オフの場合、右方向にスライド・・・前のページ。左方向にスライド・・・次のページ。


*重力*

  加速度センサでページをめくります。
  iPod Touchを長軸を中心に回転し、もどすとページをめくります。

  右を持ち上げると次、左を持ち上げると前です。


左上を押すことでリストに戻ります。


本体を回転させるとイメージも回転します。
また、操作ボタンも回転します。


== 機能 ==

しおりは自動的に挟みます。終了時に保存します。

ファイルリストの、

  緑の丸　読んでいない

  赤い丸　読んでいる途中

  白い丸　読み終わった

と言う意味です。

= 保存先 =

zipファイルは/private/var/root/Media/Comic/においてください。
フォルダに日本語名を用いたり、階層をたくさん掘っても大丈夫です。

（1.1.3以降は/private/var/mobile/Media/Comic/においてください。）


-表示-
・バウンス
	スクロールしたときに跳ね返るかどうかです。

・次ページで右上に行く
	表示位置を固定したい場合はオフにしてください。

・拡大率を維持
	オンだとページをめくっても拡大率はそのままです。

・ステータスバーを消す
	オンだと上にある時計とかの表示を消します。iComicを一旦終了させる必要があります。

・右にスライド
	オンだと右方向にスライドします。

・スクロールの速さ
	画面をなぞったときのスクロールスピードを調節します。0だと滑りません。
	80くらいだとSafariくらいのような気がします。

-操作-
・左ボタンで次のページ
	オンだと左ボタンで次のページを表示します。

・画面サイズに最適化
	オンだと表示画像を画面サイズにリサイズします。
	オフだと大きい画像(1辺1400を超えるもの）を内部で長い方の辺を1000にリサイズします。

・判定の大きさ
	ボタンの大きさです。160だと画面半分を覆うので、スクロールする場所が狭くなります。

・重力ページめくり
	加速度センサでページをめくります。
	iPod Touchを長軸を中心に回転し、もどすとページがめくられます。
	右を持ち上げると次、左を持ち上げると前です。

・ボタンページめくり
	画面の左下と右下をタップするとページがめくられます。

・スワイプページめくり
	Photoと同じ感じです。

-回転-
	Gだと重力に影響されます。
	1,2,3,4で固定表示になります。(1-正面, 2-180°, 3-左, 4-右)


※FAQ
Q1.起動しないよ？
iComicフォルダの中身はパーミッション755にしてください。
zipファイルは/private/var/root/Media/Comic/においてください。
（1.1.3以降は/private/var/mobile/Media/Comic/においてください。）


Q2.次のページに進む方法が分からない。
ReadMeくらい読めよ！


Q3.モアレが出たり、縮小が変だよ
フィルタをかけたいと思うのですが、iPod Touchではあまりにパワーが無いので無理だと思います。


Q4.読めないファイルがあるよ
画像サイズが1400とか大きすぎるものだとうまくいかないようです。
あきらめてリサイズしてください。
のはずだったのですが、リサイズ処理を追加したのでうまくいっていると思います。


Q5.ページ送りをTouch標準の写真と同じようにしてほしい
UIScrollerのdelegateについて知っていることがあったら教えてくださいな。


Q6.ファイル名の表示が変
iPodTouchのバグなのか、ファイル名に濁点・半濁点があると表示がバグるようです。
